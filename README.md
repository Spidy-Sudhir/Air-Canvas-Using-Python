Air Canvas is a fun computer-vision project where you can draw in the air using your index finger.

This project includes:
- Hand Tracking  
- Finger Gesture Recognition  
- Color Selection Buttons  
- Eraser Tool  
- Clear Canvas Feature  
- Smooth Drawing  
- Real-time Camera Feed
