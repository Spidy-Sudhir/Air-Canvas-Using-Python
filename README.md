# 🎨 Air Canvas – Draw in Air using Hand Tracking

Air Canvas is a fun computer-vision project where you can draw in the air using your index finger.  
The system detects your hand using **MediaPipe** and draws lines on a virtual canvas using **OpenCV**.

This project includes:
- Hand Tracking  
- Finger Gesture Recognition  
- Color Selection Buttons  
- Eraser Tool  
- Clear Canvas Feature  
- Smooth Drawing  
- Real-time Camera Feed  

---

## 🚀 Features

### ✍️ Drawing Mode
Raise **one finger (index)** to draw.

### ✌️ Selection Mode
Raise **two fingers (index + middle)** to select:
- Red brush  
- Blue brush  
- Green brush  
- Yellow brush  
- Eraser tool  
- Clear canvas  

### 🎨 Tools Included
| Tool | Description |
|------|-------------|
| Red | Draw with red color |
| Blue | Draw with blue |
| Green | Draw with green |
| Yellow | Highlight color |
| Eraser | Remove drawings |
| Clear | Reset full canvas |
